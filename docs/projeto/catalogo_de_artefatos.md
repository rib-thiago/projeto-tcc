# Catalogo de Artefatos

## 1. Objetivo

Este documento consolida os principais artefatos possiveis do processo,
organizados por grandes grupos. Seu objetivo e abrir o espaco de
possibilidades antes de decidir quais artefatos serao efetivamente adotados no
projeto.

## 3. Uso do catalogo

Este catalogo nao determina, por si so, que todos os artefatos devam ser
produzidos. Ele serve para:

- mapear possibilidades
- orientar selecao consciente
- evitar omissao por desconhecimento
- apoiar a definicao do conjunto efetivamente adotado

A selecao proposta para o projeto esta consolidada em
`docs/projeto/matriz_de_selecao_de_artefatos.md`.
