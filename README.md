# Projeto TCC

Este repositorio nasce como a fundacao metodologica e tecnica de um projeto de
TCC tratado com postura de produto de software e consultoria completa.

O objetivo imediato deste repositorio nao e apenas iniciar implementacao de
codigo. O objetivo imediato e consolidar:

- processo de software
- diagnostico do sistema legado
- modelagem e arquitetura
- politicas fundacionais
- estrategia operacional minima
- estrutura documental do projeto

O nucleo metodologico inicial esta concentrado em `docs/projeto/`.
